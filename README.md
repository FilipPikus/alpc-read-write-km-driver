# ALPC Read/Write Kernel-Mode Driver

> [!WARNING]  
> THE DRIVER IS DETECTED.

## Description

This is a Windows kernel-mode driver that uses ALPC/APC-based communication.  
It supports read/write operations, base address retrieval, and module information retrieval.

The project also includes KDMapper cleanup-related functionality.

## Features

- Kernel-mode read memory
- Kernel-mode write memory
- Get process base address
- Get module information
- ALPC/APC-based communication
- KDMapper cleanup-related functionality

## Educational Purpose

This repository is only a learning project.  
It is intended to help understand Windows kernel-mode driver development, driver communication methods, and low-level system programming concepts.
